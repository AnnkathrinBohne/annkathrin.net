# annkathrin.net

Static portfolio website for Annkathrin Bohne, prepared for GitHub Pages.

## Included pages

- `index.html`: Portfolio landing page
- `imprint.html`: German legal notice (Impressum)
- `privacy.html`: GDPR/EU privacy policy page
- `CNAME`: Custom domain for GitHub Pages (`annkathrin.net`)

## GitHub Pages deployment

1. Push this repository to GitHub.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (or your default branch), folder `/ (root)`
4. Save settings and wait for deployment.

If you use a custom domain (`annkathrin.net`), set it in `Settings` -> `Pages` -> `Custom domain` and configure the DNS records at your domain provider.
